# Artifact: Prose

## Target Structure
Prose review targets include:
- Documents (`.md`, `.txt`, `.doc`)
- Comments in source code
- Documentation text
- UI copy, changelog text, status text
- Any English-language text

## Target Metadata
- Lives anywhere in a repository or external
- No strict structural requirements
- Evaluated on readability, density, and style

## Content Criteria
1. **Readability** — Sentence length is appropriate and varies; clear and easy to scan
2. **Density** — Information-dense; no wordy phrases; active voice preferred; no redundant words
3. **Style** — Human-sounding, not AI-generated or templated; no AI-sounding filler; preserves user's voice
