# SMART Goal Contract

A goal states one intended outcome in a form that can be assessed.

- **Specific:** name the outcome, target subject, and scope.
- **Measurable:** name a metric, baseline when known, target value, and evidence source.
- **Achievable:** state the capability, resources, or assumptions that make the target plausible.
- **Relevant:** state why the outcome matters to the stated objective or stakeholder.
- **Time-bound:** state a deadline or review date.

Unknown baselines, evidence sources, feasibility assumptions, or deadlines must be marked as assumptions rather than invented.
