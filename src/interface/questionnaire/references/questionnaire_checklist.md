# Questionnaire Contract Checklist

Use this for conformance checks. A questionnaire contract passes when every critical item passes.

## Critical

- [ ] All questions use the "As a <role>..." form.
- [ ] Questions are prioritized (numbered) by expected impact if unaddressed.
- [ ] At least one question exists per questionnaire lens loaded.
- [ ] Each question maps to at least one source ID or persona criterion in Source Criteria.
- [ ] Questions avoid binary yes/no phrasing; they invite nuanced analysis.

## Optional but Checkable

- [ ] Base criteria questions appear early when quality/clarity is a questionnaire target.
- [ ] Source mapping covers all numbered spec IDs referenced in the contract.
