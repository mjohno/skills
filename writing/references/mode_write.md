# mode:write

Use this mode to generate polished prose from notes, rough drafts, or user input.

## Output
- A draft.
- Notes.
- Optional alternatives for the parts that matter.

## Behavior
- Infer write mode when the intent is clear.
- Ask for clarification when confidence is below 90%.
- Move through multiple drafting passes when useful.
- Preserve the user's voice when possible.
- Prefer prose that sounds human, not templated.
- Remove filler, hedging, and meta-commentary.

## Heuristics
- Optimize for clarity, density, rhythm, and natural tone.
- Bias toward technical writing conventions when the input is technical.
- Offer alternatives only where they materially help editing.
