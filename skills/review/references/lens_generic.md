# Lens: Generic (Base Lens)

This is the base lens applied to every review. It provides universal critical thinking criteria.

## Criteria

### 1. Internal Consistency
- No contradictions within the artifact
- Requirements don't conflict with each other
- Terminology is used consistently throughout

### 2. Completeness (when spec exists)
- No missing required elements defined by the spec
- All mandatory sections are present
- Cross-references are valid

### 3. Clarity
- No ambiguous language or vague requirements
- Language is precise and unambiguous
- Reader can understand the artifact without guessing intent
